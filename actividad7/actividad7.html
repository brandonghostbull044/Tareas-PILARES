Lista de definiciones

HTML
Hypertext Markup Language es un lenguaje de etiquetas que funciona como una de las piedras angulares de la Word Wide Web
Hipertexto
Se trata de una estructura no secuencial de información que permite acceder a diferentes partes de un documento de forma no lineal
Conjunto estructurado de textos, gráficos etcétera, unidos entre sí por enlaces y conecciones lógicas
Hipermedia
Conjunto estructurado de diversos medios como textos, gráficos, imágenes y sonidos unido entre sí por enlaces y conecciones lógicas para la transmisión de un información.
Lenguaje de marcado
Nos permite definir la estrcutura y el contenido de un documento mediante el uso de etiquetas
